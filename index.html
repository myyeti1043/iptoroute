IP Address Extraction Tool

Extract IPv4 and IPv6 addresses and CIDR ranges from any text, logs, or JSON format. Perfect for processing network logs, configurations, or cloud provider IP lists.

Input Example:

{"prefixes": [{"ip_prefix": "192.168.1.0/24"}, {"ip_prefix": "10.0.0.0/16"}]}
# Or plain text with IPs
192.168.1.0/24
10.0.0.0/16
172.16.0.0/12
2001:db8::/32

üí° Paste any text containing IP addresses and the tool will automatically find and extract them for you.

Network Router Configuration Generator

Convert CIDR notation or domain names (FQDN) to router configuration commands for MikroTik, Cisco, Huawei, Juniper, and Fortinet devices. Generate firewall rules, static routes, and address lists.

Input Example:

192.168.1.0/24
10.0.0.0/16
192.168.1.0/24
10.0.0.0/16
example.com

# You can also paste text with mixed IPs and domains:
"Our servers are at 192.168.1.0/24 and example.com"

üí° Select your router type and configure additional options before converting.

CIDR to IP Address and Subnet Mask Converter

Convert CIDR notation (e.g., 192.168.1.0/24) to traditional IP address with subnet mask format (192.168.1.0 255.255.255.0). Useful for legacy network configurations.

Input Example:

192.168.1.0/24
10.0.0.0/16

üí° Enter one CIDR subnet per line to convert to IP with netmask format.

IP Address and Subnet Mask to CIDR Converter

Convert traditional IP address with subnet mask format (192.168.1.0 255.255.255.0) to modern CIDR notation (192.168.1.0/24). Ideal for updating network configurations.

Input Example:

192.168.1.0 255.255.255.0
10.0.0.0 255.255.0.0

üí° Enter IP and netmask separated by space, one per line.
